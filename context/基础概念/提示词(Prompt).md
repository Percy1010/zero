# 提示词(Prompt)

## 一句话定义

`Prompt` 就是你说给 AI 听的话。你的问题、指令、背景信息、格式要求、限制条件，合在一起就是一个 Prompt。

## 通俗理解

### 类比 1: 点菜单

- 你说「随便来点吃的」，结果通常不可控
- 你说「来一份宫保鸡丁，微辣，少油，米饭单独装」，结果会更接近你的预期

Prompt 和点菜一样。你说得越具体，模型越容易按你的意思来。

### 类比 2: 给实习生布置任务

如果你只说「写个方案」，实习生根本不知道:

- 写给谁看
- 多长
- 什么风格
- 重点是什么

## 一个好 Prompt 通常包含什么

1. 角色设定
2. 背景信息
3. 具体任务
4. 格式要求
5. 示例
6. 约束条件

## 为什么 Prompt 会影响结果

因为模型不会读心。它只能根据你给出的文字去推测任务边界。

Prompt 模糊时，模型容易:

- 理解错任务
- 忽略重点
- 默认补上你没要求的内容
- 输出结构不稳定

## 一个实战对比

### 差的 Prompt

```text
帮我写一篇文章
```

### 更好的 Prompt

```text
# 角色
你是一个个人成长类公众号作者，文风幽默接地气。

# 任务
请写一篇题为《早起的人生，真的会开挂吗？》的公众号文章。

# 目标读者
25-35 岁职场人，经常熬夜，想改变但缺乏动力。

# 内容要求
1. 用有趣的故事开头
2. 给出 3 个可执行技巧
3. 结尾要有一句适合转发的金句
```

这个 Prompt 本质上是在帮模型减少歧义。

## 常见技巧

- `Zero-shot`
- `Few-shot`
- `Chain of Thought`
- `Role Prompting`
- `Step-by-step`

## 你现在只需要记住什么

- Prompt 不是神秘咒语，而是任务说明书
- 好 Prompt 的核心不是花哨，而是清楚
- 说清楚角色、任务、背景、约束和输出格式，效果会稳定很多
