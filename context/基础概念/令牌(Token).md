# 令牌(Token)

## 一句话定义

`Token` 是模型处理文本时使用的基本单位，也是很多 AI 服务计费和计算上下文长度的基础单位。

## 通俗理解

### 类比 1: 游戏币

- 你发给模型的话，会消耗输入 Token
- 模型回给你的话，会消耗输出 Token
- 问题越复杂、回答越长，消耗越多

所以 Token 既像「阅读成本」，也像「生成成本」。

### 类比 2: 出租车计价器

对话开始就像上车，模型开始处理就是起步价。你说得越多、它答得越长、上下文越复杂，计价器就走得越快。

## Token 到底是什么

Token 不完全等于「一个字」或「一个词」。

## 为什么 Token 很重要

它同时决定两件很现实的事:

1. 成本
2. 上下文窗口

## 怎么省 Token

1. Prompt 一次说清楚
2. 控制输出长度
3. 压缩无关 Context
4. 用摘要代替完整历史
5. 用合适的模型

## 你现在只需要记住什么

- Token 是模型处理文本和计费的基本单位
- AI 贵不贵、能看多长内容，都和 Token 直接相关
- 很多工程优化，本质上都是在「让有限的 Token 更值钱」
