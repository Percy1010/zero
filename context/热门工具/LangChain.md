# LangChain

## 一句话定义

`LangChain` 是一个专门用来开发 LLM 应用和 Agent 的编程框架。它提供了一整套现成模块，帮助开发者更快把 AI 应用搭起来。

## 为什么它不放在「基础概念」

因为 `LangChain` 不是像 `LLM`、`Prompt`、`RAG` 那样的通用概念，它是一个具体的工程框架。

更准确地说:

- 基础概念回答「你需要理解什么」
- LangChain 回答「程序员准备怎么把这些东西写成系统」

所以它更适合放在「热门工具 / 工程框架补充」。

## 通俗理解

### 类比 1: 乐高积木盒

如果从零做一个 AI 应用像「徒手搭房子」，LangChain 更像一盒预制好的乐高积木:

- 有连接模型的积木
- 有 Prompt 模板积木
- 有 Memory 积木
- 有 RAG 积木
- 有 Tool 积木
- 有 Agent 积木

### 类比 2: 菜谱框架

LangChain 的作用，差不多就是给程序员提供「AI 应用的菜谱和组件库」。

## 它的核心模块

- `Chain`
- `Agent`
- `Retriever`
- `Tool`
- `Memory`

## 你现在只需要记住什么

- LangChain 是框架，不是基础概念
- 它主要服务于程序员和 AI 应用开发
- 了解它有助于你读懂很多 AI 工程资料
