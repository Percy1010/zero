# Community

- [GitHub repository](https://github.com/Percy1010/zero)
- GitHub Issues: questions and bug reports
- GitHub Discussions: workflow sharing and topic proposals

## Current status

WeChat and Knowledge Planet channels are not open yet. We currently use GitHub first so discussions remain searchable and reusable.

## Suggested question template

1. Scenario: what task are you trying to finish.
2. Attempts: what you already tried.
3. Expectation: what kind of help you need (idea/prompt/tool choice/code fix).

Suggestion: feed frequently asked community questions back into docs.
