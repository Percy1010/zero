# Field Notes

Record real mistakes and fixes to build reusable experience.

## Recommended structure

- Symptom: what failed in the output.
- Trigger: what prompt/input/process caused it.
- Fix: what was changed (prompt params, tool setup, workflow step).
- Rule: what to keep as a repeatable guardrail.

## Common pitfalls

1. No acceptance criteria before generation.
2. Context is too long, key constraints get diluted.
3. AI output is treated as fact without verification.
