# Quick Start

This is the starting page for the tutorial area. You do not need to understand every tool and term at once. Start with the reading order first.

## Suggested Order

1. Start with [Core Concepts](/en/ai/concepts/).
2. Move to [Popular Tools](/en/ai/tools/).
3. Then read [Practice Notes](/en/ai/methodology/).
4. Finally use [Project Cases](/en/ai/cases/) to see real applications.

## Who This Page Is For

- Beginners who want a clear AI learning path
- Readers who get overwhelmed by too many tools at once
- Learners who prefer concept first, tools second
