# Resources

Recommended template: resource name, scenario, example, effect, caveats.

## Suggested format

1. Resource name and link.
2. Applicable scenario (who should use it and for what task).
3. Minimal example (one prompt or one concrete action).
4. Observed effect (what was improved or saved).
5. Caveats (limits, cost, and risk).

## Quality bar

- No pure marketing content.
- Prefer resources with real output examples.
- Include alternatives whenever possible to avoid lock-in.
