# Context

`Context` is the information the model can currently see and use while generating an answer.

## What context is

It is the current working set of information available to the model.

## A useful analogy

Think of context as the documents currently open on the desk.

## What context may include

- the current prompt
- previous messages
- system instructions
- uploaded files
- retrieved passages
- tool outputs

## Context vs memory

- Context is what the model sees now
- Memory is what the system stores and can reuse later

## What to remember

- Many bad answers are actually context problems
- Better context often helps more than rephrasing the same question
