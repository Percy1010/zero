# Workflow

A `Workflow` is a sequence of steps arranged to complete a task.

## What a workflow is

Instead of asking one question and getting one answer, you break the task into several stages.

## A useful analogy

It is like a standard operating procedure in an office:

1. receive the request
2. gather material
3. draft the result
4. review it
5. deliver it

## Why workflows matter in AI

Real tasks are rarely solved well in a single shot.

## Workflow vs agent

- Workflow is usually more fixed
- Agent is usually more dynamic

## What to remember

- Good AI usage is often about better workflow design, not only better prompting
