# Memory

`Memory` in AI usually means information the system chooses to keep and reuse later.

## What memory is

If context is what is visible right now, memory is what gets saved for future use.

## A useful analogy

Context is what is on the desk. Memory is what is written down in a notebook for later.

## Why memory matters

It can preserve:

- user preferences
- stable project rules
- repeated working habits

## What to remember

- Memory is not just chat history
- Good memory is selective, not endless
