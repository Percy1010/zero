# Prompt

A `Prompt` is the instruction you give to an AI system.

## What a prompt is

The easiest way to think about it is as a task brief.

## A useful analogy

If the model is an intern, the prompt is the work order you hand over.

## What a prompt usually includes

1. role
2. task
3. background
4. constraints
5. output format
6. success criteria

## Why prompts matter

Models do not read minds. They infer the task from the words you provide.

## What to remember

- A prompt is a task definition, not a magic spell
- The goal is to reduce ambiguity
