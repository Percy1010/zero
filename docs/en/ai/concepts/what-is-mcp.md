# Model Context Protocol (MCP)

`MCP` stands for `Model Context Protocol`.

## What MCP is

The easiest way to understand MCP is as a standard way for AI systems to connect to external tools and data sources.

## A useful analogy

Anthropic compares MCP to `USB-C` for AI tools. That analogy works because a shared connector is easier than inventing a different connector for every device.

## Why MCP matters

Without a shared protocol:

- every tool needs custom integration
- formats differ from system to system
- reuse becomes harder

## MCP vs function calling

- Function calling is about requesting an external capability
- MCP is about standardizing how those capabilities are connected and exposed

## What to remember

- MCP is about standardization
- It does not replace the model
- It makes tool integration cleaner and more reusable
