# Large Language Model (LLM)

A `Large Language Model`, or `LLM`, is an AI model designed to understand and generate language.

## What an LLM is

The simplest way to understand it is this: an LLM is a very large system trained to predict what language should come next based on the words it has already seen.

## A useful analogy

You can think of an LLM as someone who has read an enormous amount of writing and become very good at recognizing patterns in language.

## What an LLM can do

- draft writing
- summarize information
- translate text
- answer questions
- explain or generate code

## What an LLM does not guarantee

- factual correctness
- real-time information
- automatic access to your files or tools

## Why this concept matters

Many later concepts depend on it:

- Prompt
- Context
- RAG
- Agent
- MCP
- Function Calling

## What to remember

- An LLM is the language brain of many AI products
- It generates language well, but it is not the same as a database or search engine
