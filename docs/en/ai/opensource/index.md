# Open Source

Selection criteria: reproducible, well documented, actively maintained.

## Evaluation dimensions

1. Runnable: official example works within 30 minutes.
2. Maintainable: recent commits or responsive issues in the last 90 days.
3. Transferable: example can be adapted to your own workflow.

## Lightweight trial flow

- Step 1: run the official demo without modification.
- Step 2: replace input with your smallest real task.
- Step 3: record cost and stability before long-term adoption.
