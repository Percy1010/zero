# 搜索(Search)

## 一句话定义

`Search` 就是让 AI 在回答前先去外部信息源里找资料，弥补模型知识有截止日期、也不一定知道当前具体细节的缺陷。

## 通俗理解

没有 Search 的模型更像闭卷考试，只能靠自己「脑子里记住的内容」答题。  
有了 Search 之后，它就变成了开卷考试。

## 你现在只需要记住什么

- Search 是给模型补外部资料的能力
- 它本身不等于最终答案
- 很多联网 AI，本质上是「先搜，再写」
