# 检索增强生成(RAG)

## 一句话定义

`RAG` 是 `Retrieval-Augmented Generation` 的缩写，中文通常叫「检索增强生成」。它的核心做法是: 回答前先查你指定的资料库，再基于查到的内容来生成答案。

## 什么是「向量」

可以先把向量理解成「文字的语义指纹」。

## 你现在只需要记住什么

- RAG 的核心是「先检索，再生成」
- 它是给模型补私有知识和可靠资料的常见方案
- 它不能替代模型，但能显著降低纯靠猜的风险
