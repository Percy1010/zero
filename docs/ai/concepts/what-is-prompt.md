# 提示词(Prompt)

## 一句话定义

`Prompt` 就是你说给 AI 听的话。你的问题、指令、背景信息、格式要求、限制条件，合在一起就是一个 Prompt。

## 通俗理解

- 你说「随便来点吃的」，结果通常不可控
- 你说「来一份宫保鸡丁，微辣，少油，米饭单独装」，结果会更接近你的预期

Prompt 和点菜一样。你说得越具体，模型越容易按你的意思来。

## 一个好 Prompt 通常包含什么

1. 角色设定
2. 背景信息
3. 具体任务
4. 格式要求
5. 示例
6. 约束条件

## 你现在只需要记住什么

- Prompt 不是神秘咒语，而是任务说明书
- 好 Prompt 的核心不是花哨，而是清楚
- 说清楚角色、任务、背景、约束和输出格式，效果会稳定很多
