# 大语言模型(LLM)

## 一句话定义

`LLM` 是 `Large Language Model` 的缩写，中文叫「大语言模型」。你可以先把它理解成一个超大规模的文字预测机器: 你给它一段文字，它会根据前面的内容去预测下一个最可能出现的词或字。

## 通俗理解

### 类比 1: 超级学霸做完形填空

这就是 LLM 最核心的能力: 它先是「会接龙」，然后因为接龙能力极强，表现得像是在理解和思考。

### 类比 2: 被关在小黑屋里的天才

- 他读过非常多的资料
- 他很会组织语言
- 但他看不到外面的实时世界
- 也不会自己去操作工具

## 它能做什么

- 对话聊天
- 写作和改写
- 总结长文
- 翻译
- 写代码和解释代码

## 它不能天然做什么

- 不能自动联网获取最新信息
- 不能保证每次都说真话
- 不能天然访问你的数据库、文件和业务系统
- 不能跨对话长期记住你，除非系统额外做了 Memory

## 为什么它是后续概念的起点

- `Prompt` 解决「怎么把任务说清楚」
- `Context` 解决「模型回答时看到了什么」
- `Memory` 解决「重要信息怎么延续」
- `Search` 解决「怎么拿到最新公开信息」
- `RAG` 解决「怎么拿到你的私有资料」
- `Function Calling` 解决「怎么让模型请求调用工具」
- `MCP` 解决「怎么把外部工具统一接进来」

## 你现在只需要记住什么

- LLM 先是一个强大的文字预测系统
- 它因为预测能力极强，才表现得像会理解和思考
- 它擅长语言，不等于它天然会联网、会记忆、会操作工具
