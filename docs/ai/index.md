# 快速开始

这是整个教程区的起点。你不需要一开始就把所有工具和术语都搞懂，先知道学习顺序就够了。

## 建议从这里开始

1. 先看 [基础概念](/ai/concepts/)，把最常见的 AI 名词先讲明白。
2. 再看 [热门工具](/ai/tools/)，开始接触具体工具怎么用。
3. 然后读 [实践总结](/ai/methodology/)，理解更稳定的使用方法和协作思路。
4. 最后结合 [项目案例](/ai/cases/) 看真实场景怎么落地。

## 这一页适合谁

- 从来没有系统学过 AI 的新手
- 害怕一上来就看到太多术语和工具的人
- 想先建立一条清晰学习路径的人

## 学习时先记住一条原则

先把概念讲清楚，再学工具；先把基础操作跑通，再看高阶用法。
