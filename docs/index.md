## Zero 是什么

Zero 是一个面向职业人士的 AIGC 教程站，重点解决三个问题：零基础如何入门 AI、常见工具如何真正用起来、AI 输出如何进入可交付的工作流。

## 适合哪些人

- 想系统学习 AIGC、但不想被碎片化信息带着走的初学者
- 需要把 AI 用到文档、调研、写作、编码等真实任务里的职场用户
- 希望建立 Prompt 方法论、案例库和复盘习惯的长期学习者

## 你可以在这里学到什么

- [概念篇](/ai/concepts/)：理解 AIGC、Prompt、模型能力边界等基础问题
- [工具篇](/ai/tools/)：从具体工具入手，覆盖写作、文档、编码等使用场景
- [方法论](/ai/methodology/)：沉淀可复用的提示词结构、协作流程和验收标准
- [案例篇](/ai/cases/) 与 [经验实录](/ai/logs/)：看真实任务怎么拆解、失败后怎么修正
- [开源项目](/ai/opensource/) 与 [资源推荐](/ai/resources/)：筛选更值得投入时间的项目和资料

## 建议阅读顺序

1. 先看 [概念篇](/ai/concepts/) 建立共识。
2. 再进入 [工具篇](/ai/tools/) 找到与你任务最接近的工具。
3. 用 [方法论](/ai/methodology/) 补足 Prompt、验收和协作框架。
4. 最后结合 [案例篇](/ai/cases/) 与 [资讯日报](/news/) 持续更新认知。
