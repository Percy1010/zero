# 社媒交流

- [GitHub 仓库](https://github.com/Percy1010/zero)
- GitHub Issues：用于提问与 bug 反馈
- GitHub Discussions：用于经验交流与选题建议

## 当前状态

微信群与知识星球仍在筹备中，现阶段优先使用 GitHub 保证讨论可追溯、可沉淀。

## 建议提问模板

1. 场景：你要完成什么任务。
2. 现状：你已经尝试过什么。
3. 期望：你希望得到哪类帮助（思路/Prompt/工具选择/代码修复）。

建议：把社群高频问题回流到站内文档，持续完善知识库。
